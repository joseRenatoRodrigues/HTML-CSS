Principais formatações

Negrito / Destaque

Basta adicionar a TAG STRONG(com semântica)

Itálico / Ênfase

Basta adicionar a TAG EM(com semântica)

Texto marcado (famoso marcador de texto)

Use a TAG MARK para marcar + style="background-color + color"

Texto pequeno

Use a TAG SMALL

Texto deletado

Para textos que devem ser lidos mas desconsiderados use a TAG DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, use a TAG INS

Textos sobrescritos

por exemplo: x20+3. Use a TAG SUP

Textos subscritos

por exemplo: h2O. Use a TAG SUB

Lembre-se! Todas as TAGS são minúsculas!!!!
